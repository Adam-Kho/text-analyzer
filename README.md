# text-analyzer
This program is done when it can read text files, extract words, count their frequenciesm and display statistics.
Flow:
1. Program starts
2. Read command line input
3. Determine whether input is file or directory
4. Open file(s)
5. Process text
6. Update word count
7. Print results
8. Program ends

- Input handling
- File reading
- Text parsing
- Word normalization
- Counting
- Output formating
